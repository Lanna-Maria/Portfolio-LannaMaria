## ⚙ Como executar

Clone o projeto

```bash
git clone https://github.com/Lanna-Maria/Portfolio-LannaMaria.git
```

Entre na pasta

```bash
cd Portfolio-LannaMaria
```

Instale as dependências

```bash
npm install
```

Execute

```bash
npm run dev
```

Build

```bash
npm run build
```
